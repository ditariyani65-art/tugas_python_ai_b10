# Kesimpulan Tugas 4, 5, dan 6

### Tugas 4: Dasar Pemrograman & Struktur Data
Memahami dasar-dasar Python seperti variabel, tipe data, dan struktur data (List, Dictionary), serta logika pengkondisian dan perulangan.

### Tugas 5: Fungsi & Manipulasi String
Belajar membuat fungsi (`def`) agar kode lebih rapi dan mempelajari cara mengolah teks (string) secara efektif.

### Tugas 6: Library Data Science & OOP
Mengintegrasikan NumPy untuk statistik, Pandas untuk manajemen tabel data (DataFrame), dan konsep Class (OOP) untuk membungkus logika program menjadi objek yang terorganisir.
